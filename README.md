## Towers of Hanoi
How to run: \
Open a terminal in the project directory. \
Run either of the following commands: \
make \
make all \
Run a new command: \
./hanoi -n x -r/i \
-n x: Specifies the number of discs to be used in the puzzle. If omitted, defaults to 3 discs.\
-r/i: Specifies whether a recursive or iterative solution should be used.
